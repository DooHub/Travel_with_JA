아래는 요청하신 내용을 표로 정리한 것입니다.

### 1. 대중교통 사용법 및 비용

| 교통수단 | 사용법 | 비용 | 웹사이트 |
|----------|--------|------|----------|
| **메트로** | - 1회용 티켓: "Ticket t+"를 구매하여 탑승<br>- 10회용 티켓: "Carnet" 10장 묶음으로 구매 가능<br>- 지하철 노선도는 역 내에서 확인 | - 1회용: 약 1.90€<br>- 10회용: 약 16.90€ | [RATP 메트로 정보](https://www.ratp.fr/en/ratp) |
| **버스** | - 메트로와 동일한 "Ticket t+"로 탑승<br>- 버스 노선도는 버스 정류장에서 확인 | - 1회용: 약 1.90€<br>- 10회용: 약 16.90€ | [RATP 버스 정보](https://www.ratp.fr/en/ratp) |
| **RER (지역 철도)** | - 메트로와 동일한 티켓 사용 (RER 지역에 따라 추가 요금 발생 가능)<br>- 역에서 티켓 구매 | - 1회용: 약 1.90€ ~ 7.60€ (거리와 구간에 따라 다름) | [SNCF RER 정보](https://www.sncf.com/en) |
| **택시** | - 택시를 잡으려면 길가에서 손을 들어 택시를 호출하거나 택시 승차장을 이용<br>- 미터기로 요금 계산 | - 기본 요금: 약 7€ + 거리 요금 (킬로미터당 약 1.05€) | [택시 요금 정보](https://www.paris-taxis.com/) |
| **자전거/전동 스쿠터** | - "Vélib'" 자전거 대여소에서 대여<br>- 전동 스쿠터는 여러 앱을 통해 대여 가능 | - 자전거: 첫 30분 무료, 이후 1시간당 약 1.70€<br>- 전동 스쿠터: 1회 대여 약 1.00€, 추가 요금은 분당 약 0.25€ | [Vélib' 정보](https://www.velib-metropole.fr/) |

### 2. 식당 이용 시 주의 사항

| 주의 사항 | 설명 |
|------------|------|
| **팁 문화** | - 프랑스 식당에서는 기본적으로 팁이 포함되어 있음 (계산서에 Service compris 표시)<br>- 서비스에 만족할 경우 추가로 5-10% 정도 팁을 주는 것이 좋음 |
| **웨이터 호출** | - 웨이터를 호출할 때는 눈을 맞추고 손을 살짝 흔드는 것이 일반적<br>- "Excusez-moi" (실례합니다)라고 말하며 요청하는 것이 예의 |
| **예약** | - 인기 있는 식당은 미리 예약하는 것이 좋음<br>- 예약 없이 방문 시 대기 시간이 길어질 수 있음 |

### 3. 인종 차별 및 안전 주의 사항

| 주의 사항 | 설명 |
|------------|------|
| **인종 차별** | - 프랑스는 대체로 다양성을 존중하지만, 일부 관광 지역에서는 인종 차별을 경험할 수 있음<br>- 불쾌한 상황이 발생할 경우, 당국에 신고하거나 경찰에 도움 요청 가능 |
| **소매치기** | - 관광 명소나 붐비는 곳에서는 소매치기에 주의<br>- 가방과 소지품을 항상 주의 깊게 관리 |
| **안전** | - 불필요한 밤 늦은 외출 자제<br>- 비상 상황 시 112로 응급 연락 가능 |
